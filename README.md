# Финансовый анализ топ-10 банков России (2020-2023)

## О проекте
Анализ финансовой устойчивости крупнейших российских банков в период санкций 2022 года 
и повышения ключевой ставки ЦБ с 7.5% до 21%.

**Ключевой вопрос:** Какие банки лучше адаптировались к санкциям и высоким процентным ставкам — и почему?

## Банки
Сбербанк, ВТБ, Альфа-Банк, Газпромбанк, Тинькофф, Россельхозбанк, МКБ, Открытие, Совкомбанк, Росбанк

## Метрики
- **ROE** — рентабельность капитала
- **ROA** — рентабельность активов  
- **NIM** — чистая процентная маржа
- **NPL** — доля просроченных кредитов

## Стек
- **PostgreSQL** — хранение и обработка данных
- **Python** (pandas, psycopg2) — сбор и загрузка данных
- **SQL** — аналитические запросы (оконные функции, CTE)
- **Power BI** — интерактивный дашборд

## Источник данных
Центральный Банк РФ (cbr.ru) — Формы 101 и 102 за 2020-2023 гг.

## Ключевые выводы
1. **Сбербанк** показал наибольшую устойчивость — ROE упал с 38% до 19% в 2022, но восстановился до 39% в 2023
2. **Альфа-Банк** — лучший ROE среди частных банков (64% в 2020), адаптировался быстрее госбанков
3. **Россельхозбанк** — стабильно низкий ROE (1-5%), субсидируемая модель не зависит от рыночных условий
## Структура репозитория

**sql/** — SQL запросы
- 01_roe.sql — ROE по банкам
- 02_roa.sql — ROA по банкам
- 03_nim.sql — NIM по банкам
- 04_npl.sql — NPL ratio
- 05_roe_rank.sql — Рейтинг банков по ROE
- 06_npl_dynamics.sql — Динамика NPL до/после санкций

**dashboard/** — Power BI дашборд (banks_analysis.pbix)

**data/** — Исходные данные (banks_raw.csv)

1. Установить PostgreSQL, создать базу `banks_analysis`
2. Запустить скрипты загрузки данных
3. Открыть `banks_analysis.pbix` в Power BI Desktop
